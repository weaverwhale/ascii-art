# ASCII Art Generator

A web app that converts images into ASCII art.

## Features

- Upload any image and convert it to ASCII art
- Adjustable character density and size
- Real-time preview
- Copy or download the ASCII output

## Usage

1. Open the app in your browser
2. Upload an image or drag and drop
3. Adjust settings to your preference
4. View your ASCII art!

## Development

```bash
npm install
npm run dev
```

Built with React, TypeScript, and Vite.
